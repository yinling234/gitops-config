# GitOps 配置仓库

此仓库包含所有 Kubernetes 资源配置，由 ArgoCD 管理。

## 目录结构
- `base/`: 基础配置，包含所有环境共享的资源配置
- `overlays/`: 环境特定的配置覆盖
  - `dev/`: 开发环境配置
  - `staging/`: 预发环境配置
  - `prod/`: 生产环境配置
- `argocd/`: ArgoCD 应用定义

## 使用方法
1. 修改配置后推送到主分支
2. ArgoCD 会自动检测变更并同步到集群
3. 查看同步状态: `argocd app get myapp-dev`

## 环境差异
- **开发环境**: 1副本，低资源限制，调试日志级别
- **预发环境**: 2副本，自动扩缩容，TLS证书
- **生产环境**: 3副本，高资源限制，网络策略，手动同步
